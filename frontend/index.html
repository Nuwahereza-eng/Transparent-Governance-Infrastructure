Built for accountability — by design.

Six core systems that make corruption costly, slow, and visible.

AI Risk Engine

Each bid is scored 0–100 with a plain-English explanation: price deviation, contractor concentration, anomalies.

Hash-Chained Audit Log

Every tender, bid, approval and award is appended to a SHA-256 chain. One tampered byte breaks the entire chain.

4-Stage Approval Workflow

Department → financial → compliance → final review. No single person can push a contract through alone.

Citizen Oversight

Anyone — anonymously if needed — can report concerns. Moderator actions are all recorded on-chain.

Budget Transparency

Track allocation → release → spend per contract. Diversion is hard when every transaction is published.

Open Data Portal

Every dataset (tenders, bids, contracts, audit, contractors) is downloadable as CSV or JSON.
