Open data portal

Independent oversight starts with open data. Every public dataset is available as CSV or JSON, ready to import into Excel, Google Sheets, Python, R, or any BI tool.

Programmatic access

curl -O http://localhost:8000/api/opendata/tenders.csv
curl -O http://localhost:8000/api/opendata/contracts.json
curl    http://localhost:8000/api/opendata/audit.json
